12 # A ParamComposite object can parse a parameter file block and stores its
13 # contents in a form that allows particular parameters to be accessed and
14 # modified.
15 #
16 # A Parameter object holds the label and value(s) of a particular parameter.
17 #
18 #------------------------------------------------------------------------------
19 # Usage:
20 #
21 # To read a parameter file, use the global function readParamFile().
22 # This reads the file and returns an instance of the ParamComposite class
23 # that contains the contents of the file. For example, to read a parameter
24 # file named "param":
25 #
26 # McSimulation = readParamFile("param")
27 #
28 # This returns a ParamComposite object, which we have named McSimulation,
29 # that contains the contents of the file stored as an object tree.
30 #
31 # ------------------------------------------------------------------------------
32 # ParamComposite:
33 #
34 # An initialized ParamComposite object contains a list of children that
35 # each can be instances of Parameter or other ParamComposite subobjects.
36 # Neither the beginning line, containing the label and opening bracket,
37 # nor the closing line that contains a closing bracket are included
38 # as children. The string representation of a ParamComposite, as defined
39 # by the __str__() method, reproduces the block of lines from which it
40 # was formed.
41 #
42 # The children of a ParamComposite can be accessed in either of two ways:
43 #
44 # i) The children_ attribute is a list containing all of the children,
45 # indexed in the order in which they appear. The index starts from 0.
46 #
47 # ii) Each child is also stored as an attribute with a name given by
48 # the label of the child ParamComposite or Parameter. In a parameter
49 # file for an McSimulation,
50 #
51 # parameter = McSimulation.FileMaster.inputPrefix
52 #
53 # will thus the variable parameter a Parameter object containing the
54 # label and value for the inputPrefix parameter of the FileMaster
55 # within a McSimulation.
56 #
57 # ------------------------------------------------------------------------------
58 # Parameter:
59 #
60 # A Parameter object stores a parameter label in an attribute label_ and
61 # stores the text string representations of one or more parameter values.
62 # The internal format is flexible enough to store a single value, a
63 # single line containing several values, or multiple lines of data.
64 # All values are stored as strings: The Parameter class has no way
65 # to determine if a value should be interpreted as a double, int,
66 # string, enumeration, or something else.
67 #
68 # 1) If the parameter contains only one value, it can be returned by
69 # the value() method, with no arguments. For example:
70 #
71 # print McSimulation.FileMaster.inputPrefix.value()
72 #
73 # will print the value of the inputPrefix string.
74 #
75 # 2) If the parameter contains more than one value on a single line,
76 # the string value is returned by the method value(i), where i is
77 # the index of a particular value, indexed from 0.
78 #
79 # 3) If the parameter contains several lines of data, a particular
80 # value is obtained by the command parameter.line(i).value(j), where
81 # i is an index of the line (or row), and j is index for a value
82 # within that line (or column). Both i and j are indexed from 0.
83 #
84 # -----------------------------------------------------------------------------
