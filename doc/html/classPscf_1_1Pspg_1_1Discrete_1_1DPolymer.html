Detailed Description

template<int D>
class Pscf::Pspg::Discrete::DPolymer< D >

Descriptor and solver for a branched polymer species (Discrete chain model).

The block-bond concentrations stored in the constituent Bond<D> objects contain the block-bond concentrations (i.e., volume fractions) computed in the most recent call of the compute function.

The phi() and mu() accessor functions, which are inherited from DPolymerTmp< Bond<D> >, return the value of phi (spatial average volume fraction of a species) or mu (chemical potential) computed in the last call of the compute function. If the ensemble for this species is closed, phi is read from the parameter file and mu is computed. If the ensemble is open, mu is read from the parameter file and phi is computed.
