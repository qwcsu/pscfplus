Detailed Description

template<int D>
class Pscf::UnitCell< D >

Base template for UnitCell<D> classes, D=1, 2 or 3.

Explicit specializations are provided for D=1, 2, and 3. In each case, class UnitCell<D> is derived from UnitCellBase<D>, and defines an enumeration named LatticeSystem of the types of Bravais lattice systems in D-dimensional space.
