\doxysubsubsection{Detailed Description}
Descriptor of a molecular species in a homogeneous mixture. 

A \mbox{\hyperlink{classPscf_1_1Homogeneous_1_1Molecule}{Homogeneous\+::\+Molecule}} has\+:


\begin{DoxyItemize}
\item An array of \mbox{\hyperlink{classPscf_1_1Homogeneous_1_1Clump}{Homogeneous\+::\+Clump}} objects
\item An overall size (volume/monomer volume)
\end{DoxyItemize}

Each \mbox{\hyperlink{classPscf_1_1Homogeneous_1_1Clump}{Clump}} has a monomer type id and a size. The size is the total volume of monomers of that type in a molecule of this species.
