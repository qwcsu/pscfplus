\subsubsection*{template$<$class Block$>$\newline
class Pscf\+::\+Polymer\+Tmpl$<$ Block $>$}

Descriptor and M\+DE solver for an acyclic block polymer. 

A Polymer\+Tmpl$<$\+Block$>$ object has arrays of Block and \mbox{\hyperlink{classPscf_1_1Vertex}{Vertex}} objects. Each Block has two propagator M\+DE solver objects. The compute() member function solves the modified diffusion equation (M\+DE) for the entire molecule and computes monomer concentration fields for all blocks.
