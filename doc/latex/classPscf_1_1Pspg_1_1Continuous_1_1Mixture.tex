\subsubsection*{template$<$int D$>$\newline
class Pscf\+::\+Pspg\+::\+Continuous\+::\+Mixture$<$ D $>$}

Solver for a mixture of polymers and solvents. 

A \mbox{\hyperlink{classPscf_1_1Pspg_1_1Continuous_1_1Mixture}{Mixture}} contains a list of \mbox{\hyperlink{classPscf_1_1Pspg_1_1Continuous_1_1Polymer}{Polymer}} and \mbox{\hyperlink{classPscf_1_1Pspg_1_1Solvent}{Solvent}} objects. Each such object can solve the single-\/molecule statistical mechanics problem for an ideal gas of the associated species in a set of specified chemical potential fields, and thereby compute concentrations and single-\/molecule partition functions. A \mbox{\hyperlink{classPscf_1_1Pspg_1_1Continuous_1_1Mixture}{Mixture}} is thus both a chemistry descriptor and an ideal-\/gas solver.

A \mbox{\hyperlink{classPscf_1_1Pspg_1_1Continuous_1_1Mixture}{Mixture}} is associated with a Mesh$<$\+D$>$ object, which models a spatial discretization mesh.

Compute concentrations. 

This function calls the compute function of every molecular species, and then adds the resulting block concentration fields for blocks of each type to compute a total monomer concentration (or volume fraction) for each monomer type. Upon return, values are set for volume fraction and chemical potential (mu) members of each species, and for the concentration fields for each \mbox{\hyperlink{classPscf_1_1Pspg_1_1Continuous_1_1Block}{Block}} and \mbox{\hyperlink{classPscf_1_1Pspg_1_1Solvent}{Solvent}}. The total concentration for each monomer type is returned in the c\+Fields output parameter.

The arrays w\+Fields and c\+Fields must each have size \mbox{\hyperlink{classPscf_1_1MixtureTmpl_aa5801f478b598a8d1d5ef312be281b19}{n\+Monomer()}}, and contain fields that are indexed by monomer type index.
