\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$int D$>$\newline
class Pscf\+::\+Pspg\+::\+Discrete\+::\+D\+Mixture$<$ D $>$}

Solver for a mixture of polymers (\mbox{\hyperlink{namespacePscf_1_1Pspg_1_1Discrete}{Discrete}} chain model). 

A Mixture contains a list of Polymer and \mbox{\hyperlink{classPscf_1_1Pspg_1_1Solvent}{Solvent}} objects. Each such object can solve the single-\/molecule statistical mechanics problem for an ideal gas of the associated species in a set of specified chemical potential fields, and thereby compute concentrations and single-\/molecule partition functions. A Mixture is thus both a chemistry descriptor and an ideal-\/gas solver.

A Mixture is associated with a Mesh$<$\+D$>$ object, which models a spatial discretization mesh.
