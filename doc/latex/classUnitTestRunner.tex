\subsubsection*{template$<$class Unit\+Test\+Class$>$\newline
class Unit\+Test\+Runner$<$ Unit\+Test\+Class $>$}

Template for a \mbox{\hyperlink{classTestRunner}{Test\+Runner}} that runs test methods of an associated \mbox{\hyperlink{classUnitTest}{Unit\+Test}}. 

A instance of Unit\+Test\+Runner$<$\+My\+Test$>$ holds an array of pointers to all of the test methods of a class My\+Test that is a subclass of \mbox{\hyperlink{classUnitTest}{Unit\+Test}}. Each such test method must return void and take zero parameters. The \mbox{\hyperlink{classUnitTestRunner_a5acaa6d285f30c481a5b9230538645c5}{add\+Test\+Method()}} method is used to register a test method with the \mbox{\hyperlink{classUnitTestRunner}{Unit\+Test\+Runner}} instantiation, by adding a pointer to a test method to this array. The \mbox{\hyperlink{classUnitTestRunner_a7ff91831fb3150b633fad2a73e6c4522}{run()}} method runs all of the registered test methods in sequence.

To run a set of unit tests one must\+:


\begin{DoxyItemize}
\item Define a subclass of \mbox{\hyperlink{classUnitTest}{Unit\+Test}},
\item Define an associated subclass of \mbox{\hyperlink{classUnitTestRunner}{Unit\+Test\+Runner}},
\item Construct a \mbox{\hyperlink{classUnitTestRunner}{Unit\+Test\+Runner}} object and call \mbox{\hyperlink{classUnitTestRunner_a7ff91831fb3150b633fad2a73e6c4522}{run()}}.
\end{DoxyItemize}
