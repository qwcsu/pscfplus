\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Array\+Iterator$<$ Data $>$}

Forward iterator for an \mbox{\hyperlink{classUtil_1_1Array}{Array}} or a C array. 

An \mbox{\hyperlink{classUtil_1_1ArrayIterator}{Array\+Iterator}} is an abstraction of a pointer, similar to an S\+TL forward iterator. The $\ast$ operator returns a reference to an associated Data object, the -\/$>$ operator returns a pointer to that object. The ++ operator increments the current pointer by one array element.

Unlike an S\+TL forward iterator, an \mbox{\hyperlink{classUtil_1_1ArrayIterator}{Array\+Iterator}} contains the address of the end of the array. The \mbox{\hyperlink{classUtil_1_1ArrayIterator_a006c952467e85734bcf2f9ddc311938a}{is\+End()}} method can be used to test for termination of a for or while loop. When \mbox{\hyperlink{classUtil_1_1ArrayIterator_a006c952467e85734bcf2f9ddc311938a}{is\+End()}} is true, the current pointer is one past the end of the array, and thus the iterator has no current value, and cannot be incremented further.

An \mbox{\hyperlink{classUtil_1_1ArrayIterator}{Array\+Iterator}} behave like a pointer to non-\/const data, and provides read-\/write access to the objects to which it points.
