\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Array\+Set$<$ Data $>$}

A container for pointers to a subset of elements of an associated array. 

An \mbox{\hyperlink{classUtil_1_1ArraySet}{Array\+Set}} is a \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}} that stores pointers to a subset of the elements of an associated \mbox{\hyperlink{classUtil_1_1Array}{Array}} container or bare C array. Pointers to the elements of this set are stored in a contiguous sequence, with indices in the range 0, ..., \mbox{\hyperlink{classUtil_1_1PArray_a2b0709ba78b786a7eb4412e817c486b1}{size()}} -\/ 1. The order in which these pointers are stored is mutable, and generally changes whenever an element is removed.

The \mbox{\hyperlink{classUtil_1_1ArraySet_a369d0413775cfab246ed105a6a103735}{append()}} method appends a pointer to a new element to the end of the sequence and increments the size. The \mbox{\hyperlink{classUtil_1_1ArraySet_a7c21a57396aa00fb4e2e755415d20a72}{remove()}} method removes a specified element, then moves the pointer of the last element to the space vacated by the removed element (unless the removed element was the last in the sequence), and decrements the size. The order in which the remaining elements of an \mbox{\hyperlink{classUtil_1_1ArraySet}{Array\+Set}} are stored thus can change whenever an element is removed.

An \mbox{\hyperlink{classUtil_1_1ArraySet}{Array\+Set}} provides O(\+N) sequential access to all elements of a set, O(1) insertion and deletion, and O(1) access to a randomly chosen element.
