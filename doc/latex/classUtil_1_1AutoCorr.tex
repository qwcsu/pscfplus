\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data, typename Product$>$\newline
class Util\+::\+Auto\+Corr$<$ Data, Product $>$}

Auto-\/correlation function for one sequence of Data values. 

This class calculates an autocorrelation function for a sequence x(i) of values of a variable or object of type Data. The resulting autocorrelation function is and array of values of type Product, where C(j) = $<$x(i-\/j), x(i)$>$. Here $<$A,B$>$ denotes an inner product of type Product for objects A and B of type Data.

The meaning of the inner product is defined for various data types b the overloaded function Product product(\+Data, Data) that is defined for double, complex and \mbox{\hyperlink{classUtil_1_1Vector}{Vector}} data in the \mbox{\hyperlink{product_8h_source}{product.\+h}} file.

The zero value for variables of type Data is returned by the overloaded function void set\+To\+Zero(\+Data) method defined in the set\+To\+Data.\+h file.
