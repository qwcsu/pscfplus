\doxysubsubsection{Detailed Description}
A cardinal B-\/spline basis function. 

A cardinal B-\/\+Spline of order m or degree k = m -\/ 1 is a piecewise continuous polynomial of degree k defined over the domain \mbox{[}0, k+1\mbox{]}. Such a function is defined by k different polynomials, each of which has a domain \mbox{[}i, i+1\mbox{]} for an integer i with 0 $<$= i $<$=k. For k $>$ 0, the function and k-\/1 derivatives are continuous.

A \mbox{\hyperlink{classUtil_1_1CardinalBSpline}{Cardinal\+B\+Spline}} object of degree k has k \mbox{\hyperlink{classUtil_1_1Polynomial}{Polynomial$<$double$>$}} objects, indexed by an integer 0 $<$= i $<$= k, each of which defines the polynomial with a domain \mbox{[}i,i+1\mbox{]}.

If object b is a \mbox{\hyperlink{classUtil_1_1CardinalBSpline}{Cardinal\+B\+Spline}} of degree k, then\+:


\begin{DoxyItemize}
\item Operator b\mbox{[}i\mbox{]} returns the \mbox{\hyperlink{classUtil_1_1Polynomial}{Polynomial$<$double$>$}} for domain \mbox{[}i,i+1\mbox{]}
\item Operator b(x) returns the value of basis function b for real x. 
\end{DoxyItemize}
