class Util\+::\+Const\+Array\+Iterator$<$ Data $>$}

Forward const iterator for an \mbox{\hyperlink{classUtil_1_1Array}{Array}} or a C array. 

A \mbox{\hyperlink{classUtil_1_1ConstArrayIterator}{Const\+Array\+Iterator}} is a forward iterator for an array of const data. It is an abstraction of a pointer to const, similar to an S\+TL const forward iterator. The $\ast$ operator returns a const reference to an associated Data object the -\/$>$ operator returns a const pointer to that object. The ++ operator increments the current pointer by one array element.

Unlike an S\+TL const forward iterator, an \mbox{\hyperlink{classUtil_1_1ConstArrayIterator}{Const\+Array\+Iterator}} contains the address of the end of the array. The \mbox{\hyperlink{classUtil_1_1ConstArrayIterator_ab640e045c46a5615f9f19a41a5dd0e14}{is\+End()}} method can be used to test for termination of a for or while loop. When \mbox{\hyperlink{classUtil_1_1ConstArrayIterator_ab640e045c46a5615f9f19a41a5dd0e14}{is\+End()}} is true, the current pointer is one past the end of the array, and thus the iterator has no current value.

A \mbox{\hyperlink{classUtil_1_1ConstArrayIterator}{Const\+Array\+Iterator}} behave like a pointer to constant data, and provides read-\/only access to the object to which it points. Use an \mbox{\hyperlink{classUtil_1_1ArrayIterator}{Array\+Iterator}} if you need read-\/write access.
