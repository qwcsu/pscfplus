\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Const\+P\+Array\+Iterator$<$ Data $>$}

Forward iterator for a \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}}. 

An \mbox{\hyperlink{classUtil_1_1ConstPArrayIterator}{Const\+P\+Array\+Iterator}} is an abstraction of a pointer, similar to an S\+TL forward iterator. The $\ast$ operator returns a reference to an associated Data object, the -\/$>$ operator returns a pointer to that object. The ++ operator increments the current pointer by one array element.

Unlike an S\+TL forward iterator, an \mbox{\hyperlink{classUtil_1_1ConstPArrayIterator}{Const\+P\+Array\+Iterator}} contains the address of the end of the array. The \mbox{\hyperlink{classUtil_1_1ConstPArrayIterator_adcb369a3888508422598f3154dbab101}{is\+End()}} method can be used to test for termination of a for or while loop. When \mbox{\hyperlink{classUtil_1_1ConstPArrayIterator_adcb369a3888508422598f3154dbab101}{is\+End()}} is true, the iterator has no current value, and cannot be incremented further. The \mbox{\hyperlink{classUtil_1_1ConstPArrayIterator_adcb369a3888508422598f3154dbab101}{is\+End()}} method returns true either if the iterator\+: i) has already been incremented one past the end of an associated \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}}, or ii) is in a null state that is produced by the constructor and the clear() method.
