\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+D\+S\+Array$<$ Data $>$}

Dynamically allocated array with variable logical size. 

A \mbox{\hyperlink{classUtil_1_1DSArray}{D\+S\+Array}} $<$ Data $>$ is a wrapper for a dynamically allocated C array, with continuous elements and a logical size that may be less than or equal to its physical capacity. The logical size is the number of contiguous elements that have been added using the \mbox{\hyperlink{classUtil_1_1DSArray_a4ac821902ea29016412aaf7d545e3fd7}{append()}} method.
