\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data, int Capacity$>$\newline
class Util\+::\+F\+S\+Array$<$ Data, Capacity $>$}

A fixed capacity (static) contiguous array with a variable logical size. 

An \mbox{\hyperlink{classUtil_1_1FSArray}{F\+S\+Array}} $<$ Data, Capacity $>$ is a wrapper for a statically allocated C array containing Capacity objects of type Data. An \mbox{\hyperlink{classUtil_1_1FSArray}{F\+S\+Array}} has both a Capacity that is set at compile time, which is the physical size of the underlying C array, and a logical size, which is the number of contiguous elements (from 0 to one less than its size) that contain valid data. The size is initialized to zero, and can only be increased only by the \mbox{\hyperlink{classUtil_1_1FSArray_a74ef93eabdb54ff1453a3252019984bc}{append()}} method, which adds a new element to the end of the array. ~\newline


When compiled in debug mode (i.\+e., when N\+D\+E\+B\+UG is defined) the subcript operator \mbox{[}\mbox{]} checks that the index is less than the logical size, and not merely less than the capacity.
