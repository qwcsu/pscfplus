\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+G\+P\+Array$<$ Data $>$}

An automatically growable \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}}. 

An \mbox{\hyperlink{classUtil_1_1GPArray}{G\+P\+Array}} is a \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}} that grows as needed as objects are appended. Like any \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}}, it holds pointers to objects, rather than objects. The associated objects are not destroyed when a \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}} is deallocated or destroyed.
