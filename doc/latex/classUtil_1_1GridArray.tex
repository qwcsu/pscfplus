\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Grid\+Array$<$ Data $>$}

Multi-\/dimensional array with the dimensionality of space. 

The memory for a \mbox{\hyperlink{classUtil_1_1GridArray}{Grid\+Array}} is stored in a single one-\/dimensional C array. The subscript \mbox{[}\mbox{]} operator is overloaded to return an element indexed by a one-\/dimensional rank, and the () operator is overloaded to return an element indexed by an \mbox{\hyperlink{classUtil_1_1IntVector}{Int\+Vector}} of grid coordinates.
