\doxysubsubsection{Detailed Description}
Wrapper for an int, for formatted ostream output. 

An \mbox{\hyperlink{classUtil_1_1Int}{Int}} object has a int numerical value, and a minimum output field width. The $<$$<$ operator for an \mbox{\hyperlink{classUtil_1_1Int}{Int}} uses the specified width. The numerical value and width may both be specified as parameters to a constructor. If the width is not specified as a constructor parameter, it is set within the constructor to a default value equal to \mbox{\hyperlink{classUtil_1_1Format_ac6cf6620e160c66757429bec0fb4dd23}{Format\+::default\+Width()}}.

An \mbox{\hyperlink{classUtil_1_1Int}{Int}} object may be passed to an ostream as a temporary object. For example, the expression\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{std::cout << \mbox{\hyperlink{classUtil_1_1Int_aeec8da6a22358888665628f09ed1eac1}{Int}}(13) << \mbox{\hyperlink{classUtil_1_1Int_aeec8da6a22358888665628f09ed1eac1}{Int}}(25, 10) << std::endl;}
\end{DoxyCode}
 outputs the number 13 using the default width, followed by the number 25 in a field of minimum width 10.
