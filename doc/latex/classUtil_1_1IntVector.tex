\doxysubsubsection{Detailed Description}
An \mbox{\hyperlink{classUtil_1_1IntVector}{Int\+Vector}} is an integer Cartesian vector. 

The Cartesian elements of a \mbox{\hyperlink{classUtil_1_1IntVector}{Int\+Vector}} can be accessed using array notation\+: The elements of a three dimensional \mbox{\hyperlink{classUtil_1_1IntVector}{Int\+Vector}} v are v\mbox{[}0\mbox{]}, v\mbox{[}1\mbox{]}, and v\mbox{[}2\mbox{]}. The subscript operator \mbox{[}\mbox{]} returns elements as references, which can be used on either the left or right side of an assignment operator.

The arithmetic assignment operators +=, -\/=, $\ast$=, and /= are overloaded. The operators += and -\/= represent increment or decrement by a vector, while $\ast$= and /= represent multiplication or division by an integer.

All other unary and binary mathematical operations are implemented as methods. Operations that yield a scalar result, such as a dot product, are implemented as methods that return the resulting value. Operations that yield a \mbox{\hyperlink{classUtil_1_1IntVector}{Int\+Vector}}, such as vector addition, are implemented by methods that assign the result to the invoking vector, and return a reference to the invoking vector. For example,

For efficiency, all methods in this class are inlined.
