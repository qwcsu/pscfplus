\doxysubsubsection{Detailed Description}
A label string in a file format. 

The operator $>$$>$ for a label checks if the expected label was found. The operator $<$$<$ outputs the expected label.

The constructor takes a parameter is\+Required that determines whether the label must be matched (is\+Required == true), or if it is optional (is\+Required == false). If the input value read by the $>$$>$ operator does not match the expected value and is\+Required is true, the $>$$>$ operator will print an error message to the \mbox{\hyperlink{classUtil_1_1Log_a7b86f7c53e482f8f4a1a8b756c162817}{Log\+::file()}} and then throw an \mbox{\hyperlink{classUtil_1_1Exception}{Exception}}. If the input value does not match and is\+Required is false, the $>$$>$ operator stores the input value in a string buffer, and will compare it to subsequent values until a match is found.
