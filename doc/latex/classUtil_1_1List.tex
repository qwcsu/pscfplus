class Util\+::\+List$<$ Data $>$}

Linked list class template. 

This list implementation is based on an underlying C array of Node$<$\+Data$>$ objects. This array may be used by several \mbox{\hyperlink{classUtil_1_1List}{List}} objects, and so must be allocated outside the Link class and provided via the initialize method.
