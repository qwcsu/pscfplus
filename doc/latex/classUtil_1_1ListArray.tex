\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+List\+Array$<$ Data $>$}

An array of objects that are accessible by one or more linked \mbox{\hyperlink{classUtil_1_1List}{List}} objects. 

A \mbox{\hyperlink{classUtil_1_1ListArray}{List\+Array}} is an allocatable array of data objects that also provides access to some or all of its via one or more associated \mbox{\hyperlink{classUtil_1_1List}{List}} objects. Each element of the array may be part of at most one \mbox{\hyperlink{classUtil_1_1List}{List}}.
