\doxysubsubsection{Detailed Description}
Wrapper for a long int, for formatted ostream output. 

An \mbox{\hyperlink{classUtil_1_1Lng}{Lng}} object has a long int numerical value, and a minimum output field width. The $<$$<$ operator for an \mbox{\hyperlink{classUtil_1_1Lng}{Lng}} uses the specified width. The numerical value and width may both be optionally specified as parameters to a constructor. If the width is not specified, it is is is set to a default value equal to \mbox{\hyperlink{classUtil_1_1Format_ac6cf6620e160c66757429bec0fb4dd23}{Format\+::default\+Width()}}.

An \mbox{\hyperlink{classUtil_1_1Lng}{Lng}} object may be passed to an ostream as a temporary object. For example, the expression\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{std::cout << \mbox{\hyperlink{classUtil_1_1Lng_a922ecce0af05ff8fc68b568923110c2d}{Lng}}(13) << \mbox{\hyperlink{classUtil_1_1Lng_a922ecce0af05ff8fc68b568923110c2d}{Lng}}(25, 10) << std::endl;}
\end{DoxyCode}
 outputs the number 13 using the default width, followed by the number 25 in a field of minimum width 10.
