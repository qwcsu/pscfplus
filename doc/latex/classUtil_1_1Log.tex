\doxysubsubsection{Detailed Description}
A static class that holds a log output stream. 

The \mbox{\hyperlink{classUtil_1_1Log}{Log}} class has one a static pointer member that points to an ostream that should be used by all other classes to output log and error messages. This stream is accessed by the \mbox{\hyperlink{classUtil_1_1Log_a7b86f7c53e482f8f4a1a8b756c162817}{file()}} method.

The log file initialized to point to std\+::cout. It may be reset to point to a ofstream file object using the static \mbox{\hyperlink{classUtil_1_1Log_a4d40ebcfa31f857e1cd57734419dbbde}{set\+File()}} method.
