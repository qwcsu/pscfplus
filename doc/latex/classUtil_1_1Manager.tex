\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Manager$<$ Data $>$}

Template container for pointers to objects with a common base class. 

A Manager$<$\+Data$>$ has an array of Data$\ast$ pointers to Data objects, an array of corresponding subclass names, and a pointer to a Factory$<$\+Data$>$ object. The default implementation of the \mbox{\hyperlink{classUtil_1_1Manager_a115f9d62a9e538f20660683eb0eb3a10}{Manager$<$\+Data$>$\+::read\+Param()}} method uses an associated Factory$<$\+Data$>$ object to recognize the class name string that begins a polymorphic block in a parameter file (which must refer to a known subclass of Data) and to instantiate an object of the specified subclass.

Subclasses of Manager$<$\+Data$>$ are used to manage arrays of Species, Mc\+Move, and Analyzer objects.

Read and create a set of objects. 

The default implementation of this method reads a sequence of blocks for different subclasses of Data, terminated by a closing bracket. For each block it\+:


\begin{DoxyItemize}
\item reads a class\+Name string for a subclass of Data,
\item uses factory object to create a new instance of class\+Name.
\item invokes the \mbox{\hyperlink{classUtil_1_1Manager_a115f9d62a9e538f20660683eb0eb3a10}{read\+Param()}} method of the new object.
\end{DoxyItemize}

The implementation of the factory must recognize all valid class\+Name string values, and invoke the appropriate constructor for each. The loop over blocks terminates when it encounters a closing bracket \textquotesingle{}\}\textquotesingle{} surrounded by white space.
