\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Matrix$<$ Data $>$}

Two-\/dimensional array container template (abstract). 

An \mbox{\hyperlink{classUtil_1_1Matrix}{Matrix}} object A is a two-\/dimensional array in which the operator A(i, j) returns a reference to element in column j of row i.

The memory for a \mbox{\hyperlink{classUtil_1_1Matrix}{Matrix}} is stored in a single one-\/dimensional C array, in which each row is stored as a consecutive block.

Class \mbox{\hyperlink{classUtil_1_1Matrix}{Matrix}} is an abstract class because it cannot allocate memory. Concrete subclasses include \mbox{\hyperlink{classUtil_1_1DMatrix}{D\+Matrix}} and \mbox{\hyperlink{classUtil_1_1FMatrix}{F\+Matrix}}.
