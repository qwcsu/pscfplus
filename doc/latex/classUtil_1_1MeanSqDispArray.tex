Mean-\/squared displacement (M\+SD) vs. 

time for an ensembles of sequences.

This class calculates the mean-\/squared difference $<$$\vert$x(i) -\/ x(i-\/j)$\vert$$^\wedge$2$>$ for an ensemble of statistically equivalent sequences x(i) of values of a variable of type Data. The meaning of $\vert$a -\/ b$\vert$$^\wedge$2 is defined for int, double, and \mbox{\hyperlink{classUtil_1_1Vector}{Vector}} data by explicit specializations of the private method double sq\+Diff(\+Data\&, Data).
