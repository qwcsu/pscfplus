\doxysubsubsection{Detailed Description}
Allows information from every processor in a communicator, to be output in rank sequence. 

The \mbox{\hyperlink{classUtil_1_1MpiLogger_a5ab2113f1d384e087a41c7ba949ffce3}{begin()}} method for processor of rank $>$ 0 waits for receipt of a message from processor rank -\/ 1. The \mbox{\hyperlink{classUtil_1_1MpiLogger_a030c2339c8bf527706045445b42a6cbd}{end()}} method sends a message to processor rank + 1.
