\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+P\+Array$<$ Data $>$}

An array that only holds pointers to its elements. 

A P\+Array$<$\+Data$>$ is an array that is implemented by storing pointers to Data objects, rather than actual Data objects. The array suscript operator \mbox{[}\mbox{]} returns a reference to an associated Data object, as for Array$<$\+Data$>$. A P\+Array$<$\+Data$>$ is not responsible for destroying the associated Data objects.

A \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}} cannot be instantiated, because its constructor is protected. \mbox{\hyperlink{classUtil_1_1PArray}{P\+Array}} is a base class for \mbox{\hyperlink{classUtil_1_1DPArray}{D\+P\+Array}} and for \mbox{\hyperlink{classUtil_1_1ArraySet}{Array\+Set}}.
