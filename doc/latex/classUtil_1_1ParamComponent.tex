\doxysubsubsection{Detailed Description}
Abstract base class for classes that input and output parameters to file. 

The read\+Param method reads a parameter or parameter list from iostream. The write\+Param method writes a parameter or parameter list to an ostream. The same io format should be used by write and read methods.
