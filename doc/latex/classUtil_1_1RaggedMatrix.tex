\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Ragged\+Matrix$<$ Data $>$}

A 2D array in which different rows can have different lengths. 

A \mbox{\hyperlink{classUtil_1_1RaggedMatrix}{Ragged\+Matrix}} object A is a two-\/dimensional array in which the operator A(i,j) returns a reference to element j of row i, and in which different rows have different lengths. Class \mbox{\hyperlink{classUtil_1_1RaggedMatrix}{Ragged\+Matrix}} cannot be instantiated, and functions like an abstract base class.

The memory for a \mbox{\hyperlink{classUtil_1_1RaggedMatrix}{Ragged\+Matrix}} is stored in a one-\/dimensional C array.
