\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{classUtil_1_1Random}{Random}} number generator. 

This class provides functions that return several forms of random numbers, using an internal Mersenne-\/\+Twister random number generator. ~\newline


The generator may be seeded either by reading a seed from file, using the \mbox{\hyperlink{classUtil_1_1ParamComposite_a6fc094b057fd9a21641b089f6f063b39}{read\+Param()}} method, or by using set\+Seed() to set or reset it explicitly. In either case, inputting a positive integer causes that value to be used as a seed, but inputting a value of 0 causes the use of a seed that is generated from the system clock.

If the program is compiled with M\+PI, and M\+PI is initialized, then any automatically generated seed is also offset by a value that depends on the rank of the processor within the M\+PI world communicator, so that different processor use different seeds.
