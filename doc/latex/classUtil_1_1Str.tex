\doxysubsubsection{Detailed Description}
Wrapper for a std\+::string, for formatted ostream output. 

An \mbox{\hyperlink{classUtil_1_1Str}{Str}} object has std\+::string value, and an integer output field width. The $<$$<$ operator for an \mbox{\hyperlink{classUtil_1_1Str}{Str}} object uses the specified width. The value and width may both be specified as parameters to a constructor. If the width is not specified as a constructor parameter, it is set within the constructor to a default value given by \mbox{\hyperlink{classUtil_1_1Format_ac6cf6620e160c66757429bec0fb4dd23}{Format\+::default\+Width()}}.

An \mbox{\hyperlink{classUtil_1_1Str}{Str}} object may be passed to an ostream as a temporary object. For example, the expression\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{std::cout << \mbox{\hyperlink{classUtil_1_1Str_a4f76794713df0d02cfb747fd247282d7}{Str}}(\textcolor{stringliteral}{"Hello"}) << \mbox{\hyperlink{classUtil_1_1Str_a4f76794713df0d02cfb747fd247282d7}{Str}}(\textcolor{stringliteral}{"World"}, 20) << std::endl;}
\end{DoxyCode}
 outputs \char`\"{}\+Hello\char`\"{} using the default width, followed by \char`\"{}\+World\char`\"{} in a field of width 20.
