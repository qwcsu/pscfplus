\doxysubsubsection{Detailed Description}
Calculates averages of all components of a Tensor-\/valued variable. 

\mbox{\hyperlink{classUtil_1_1TensorAverage}{Tensor\+Average}} is a simple container for an array of \mbox{\hyperlink{classUtil_1_1Average}{Average}} objects, each of which calculates averages and error estimates for one component of a \mbox{\hyperlink{classUtil_1_1Tensor}{Tensor}}.
