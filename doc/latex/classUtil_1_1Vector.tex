\doxysubsubsection{Detailed Description}
A \mbox{\hyperlink{classUtil_1_1Vector}{Vector}} is a Cartesian vector. 

The Cartesian elements of a \mbox{\hyperlink{classUtil_1_1Vector}{Vector}} can be accessed using array notation\+: The elements of a three dimensional \mbox{\hyperlink{classUtil_1_1Vector}{Vector}} v are v\mbox{[}0\mbox{]}, v\mbox{[}1\mbox{]}, and v\mbox{[}2\mbox{]}. The subscript operator \mbox{[}\mbox{]} returns elements as references, which can be used on either the left or right side of an assignment operator.

The arithmetic assignment operators +=, -\/=, $\ast$=, and /= are overloaded. The operators += and -\/= represent increment or decrement by a vector, while $\ast$= and /= represent multiplication or division by a scalar.

All other unary and binary mathematical operations are implemented as methods. Operations that yield a scalar result, such as a dot product, are implemented as methods that return the resulting value. Operations that yield a \mbox{\hyperlink{classUtil_1_1Vector}{Vector}}, such as vector addition, are implemented by methods that assign the result to the invoking vector, and return a reference to the invoking vector. For example,

 This syntax for \mbox{\hyperlink{classUtil_1_1Vector}{Vector}} valued operations avoids dynamic allocation of temporary \mbox{\hyperlink{classUtil_1_1Vector}{Vector}} objects, by requiring that the invoking function provide an object to hold the result.

For efficiency, all methods in this class are declared inline.
