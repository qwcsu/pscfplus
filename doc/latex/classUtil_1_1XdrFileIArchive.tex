\doxysubsubsection{Detailed Description}
Loading / input archive for binary X\+DR file. 

X\+DR is a standard protocol for writing and reading binary in a portable format. This archive saves data to an associated file in X\+DR format. It depends on the unix xdr library $<$rpc/xdr.\+h$>$. Because this library is written in C (not C++), this archive uses a standard C library file handle, not a C++ iostream.
