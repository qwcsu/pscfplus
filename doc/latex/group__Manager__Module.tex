\doxysubsubsection{Detailed Description}
A \mbox{\hyperlink{classUtil_1_1Manager}{Manager}} is a container for items with a common base class.

A \mbox{\hyperlink{classUtil_1_1Manager}{Manager}} is is a subclass of \mbox{\hyperlink{classUtil_1_1ParamComposite}{Param\+Composite}} that manages a list of subclasses of a base class, which it creates using an associated \mbox{\hyperlink{classUtil_1_1Factory}{Factory}}.
