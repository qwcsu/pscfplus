\doxysubsubsection{Detailed Description}
Two-\/dimensional array containers that store by objects value. 

\mbox{\hyperlink{classUtil_1_1Matrix}{Matrix}} containers overload the () operator to return elements by refernce. If A is a matrix object A(i, j) returns element i, j of the matrix.

The \mbox{\hyperlink{classUtil_1_1Matrix}{Matrix}} base class defines a conventional matrix, in which all rows are the same length. The \mbox{\hyperlink{classUtil_1_1DMatrix}{D\+Matrix}} and \mbox{\hyperlink{classUtil_1_1FMatrix}{F\+Matrix}} subclasses use dynamically allocated and fixed-\/size arrays, respectively.

The \mbox{\hyperlink{classUtil_1_1RaggedMatrix}{Ragged\+Matrix}} base class and \mbox{\hyperlink{classUtil_1_1DRaggedMatrix}{D\+Ragged\+Matrix}} subclass define two-\/dimensional containers in which different rows can have different lengths, though the list of row dimensions can be specified only once.
