\doxysubsubsection{Detailed Description}
Classes used to read parameters from a parameter file. Any class that must read values of member variables from a file should be derived from \mbox{\hyperlink{classUtil_1_1ParamComposite}{Param\+Composite}}, which provides methods for reading and writing a parameter file, using a programmatically defined file format.

\mbox{\hyperlink{classUtil_1_1ParamComponent}{Param\+Component}} is an abstract base class. The classes \mbox{\hyperlink{classUtil_1_1ParamComposite}{Param\+Composite}}, \mbox{\hyperlink{classUtil_1_1Parameter}{Parameter}}, \mbox{\hyperlink{classUtil_1_1Begin}{Begin}}, \mbox{\hyperlink{classUtil_1_1End}{End}}, and \mbox{\hyperlink{classUtil_1_1Blank}{Blank}} are derived directly from \mbox{\hyperlink{classUtil_1_1ParamComponent}{Param\+Component}}. \mbox{\hyperlink{classUtil_1_1Parameter}{Parameter}}, \mbox{\hyperlink{classUtil_1_1Begin}{Begin}}, \mbox{\hyperlink{classUtil_1_1End}{End}}, and \mbox{\hyperlink{classUtil_1_1Blank}{Blank}} are \char`\"{}leaf\char`\"{} notes it a tree structure.

Each subclasses of \mbox{\hyperlink{classUtil_1_1Parameter}{Parameter}} represents a parameter associated with a different type of C++ object. Such subclasses include class templates \mbox{\hyperlink{classUtil_1_1ScalarParam}{Scalar\+Param}}, \mbox{\hyperlink{classUtil_1_1CArrayParam}{C\+Array\+Param}}, \mbox{\hyperlink{classUtil_1_1DArrayParam}{D\+Array\+Param}}, \mbox{\hyperlink{classUtil_1_1FArrayParam}{F\+Array\+Param}}, \mbox{\hyperlink{classUtil_1_1CArray2DParam}{C\+Array2\+D\+Param}} and Matrix\+Param. The template \mbox{\hyperlink{classUtil_1_1ScalarParam}{Scalar\+Param}} represents any parameter that is associated with either a primitive C type or a user type for which their exist overloaded \char`\"{}$<$$<$\char`\"{} and \char`\"{}$>$$>$\char`\"{} file IO operators. The templates \mbox{\hyperlink{classUtil_1_1CArrayParam}{C\+Array\+Param}}, \mbox{\hyperlink{classUtil_1_1DArrayParam}{D\+Array\+Param}}, \mbox{\hyperlink{classUtil_1_1FArrayParam}{F\+Array\+Param}}, \mbox{\hyperlink{classUtil_1_1CArray2DParam}{C\+Array2\+D\+Param}}, and Matrix\+Param difine parameter file formats for different types of 1D and 2D arrays.
