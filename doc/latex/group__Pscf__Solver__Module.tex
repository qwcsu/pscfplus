\doxysubsubsection{Detailed Description}
Templates for classes that solve modified diffusion equations. 

The templates defined in this module are designed to be used as base classes for classes that define a variety of different implementations of self-\/consistent field theory (S\+C\+FT), in which each implementation uses a particular set of algorithms to solve the modified diffusion equation (M\+DE) in a particular type of geometry.

To define an implementation of S\+C\+FT, one must define the following set of solver classes derived from these templates\+:


\begin{DoxyItemize}
\item A Propagator class, derived from \mbox{\hyperlink{classPscf_1_1PropagatorTmpl}{Propagator\+Tmpl}}
\item A Block class, derived from Block\+Tmpl$<$\+Propagator$>$
\item A Polymer class, derived from Polymer\+Tmpl$<$\+Block$>$
\item A Solvent class, derived from Solvent\+Tmpl$<$\+Block$>$
\item A Mixture class, derived from Mixture\+Tmpl$<$\+Polymer, Solvent$>$ 
\end{DoxyItemize}
