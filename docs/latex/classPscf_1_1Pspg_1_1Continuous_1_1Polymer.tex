\subsubsection*{template$<$int D$>$\newline
class Pscf\+::\+Pspg\+::\+Continuous\+::\+Polymer$<$ D $>$}

Descriptor and solver for a branched polymer species. 

The block concentrations stored in the constituent Block$<$\+D$>$ objects contain the block concentrations (i.\+e., volume fractions) computed in the most recent call of the compute function.

The \mbox{\hyperlink{classPscf_1_1Species_a1956df7c7b319005c9f8b6cc274a7dc7}{phi()}} and \mbox{\hyperlink{classPscf_1_1Species_acaa3a7b3452b1100290920467bf99a57}{mu()}} accessor functions, which are inherited from Polymer\+Tmp$<$ Block$<$\+D$>$ $>$, return the value of phi (spatial average volume fraction of a species) or mu (chemical potential) computed in the last call of the compute function. If the ensemble for this species is closed, phi is read from the parameter file and mu is computed. If the ensemble is open, mu is read from the parameter file and phi is computed.
