\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+Array$<$ Data $>$}

\mbox{\hyperlink{classUtil_1_1Array}{Array}} container class template. 

An \mbox{\hyperlink{classUtil_1_1Array}{Array}} is a sequence that supports random access via an overloaded operator \mbox{[}\mbox{]}, and that wraps a dynamically allocated C array.

\mbox{\hyperlink{classUtil_1_1Array}{Array}} is a base class for \mbox{\hyperlink{classUtil_1_1DArray}{D\+Array}}, which is dynamically allocated, and \mbox{\hyperlink{classUtil_1_1RArray}{R\+Array}}, which acts as a reference to another \mbox{\hyperlink{classUtil_1_1DArray}{D\+Array}} or \mbox{\hyperlink{classUtil_1_1FSArray}{F\+S\+Array}}.

When compiled in debug mode (i.\+e., when N\+D\+E\+B\+UG is not defined) the subscript operator \mbox{[}\mbox{]} checks the validity of the element index.
