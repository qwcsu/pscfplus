\doxysubsubsection{Detailed Description}
Wrapper for an bool value, for formatted ostream output. 

An \mbox{\hyperlink{classUtil_1_1Bool}{Bool}} object has a bool value, and a minimum output field width. The $<$$<$ operator for a \mbox{\hyperlink{classUtil_1_1Bool}{Bool}} uses the specified width. The value and width may both be specified as parameters to a constructor. If the width is not specified as a constructor parameter, it is set within the constructor to the default \mbox{\hyperlink{classUtil_1_1Format_ac6cf6620e160c66757429bec0fb4dd23}{Format\+::default\+Width()}}.

An \mbox{\hyperlink{classUtil_1_1Bool}{Bool}} object may be passed to an ostream as a temporary object. For example, the expression\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{std::cout << \mbox{\hyperlink{classUtil_1_1Bool_a4f4b0a2ef084437775e74d8129c033ff}{Bool}}(\textcolor{keyword}{true}) << \mbox{\hyperlink{classUtil_1_1Bool_a4f4b0a2ef084437775e74d8129c033ff}{Bool}}(\textcolor{keyword}{false}, 10) << std::endl;}
\end{DoxyCode}
 outputs the value true using the default width, followed by the false value in a field of minimum width 10.
