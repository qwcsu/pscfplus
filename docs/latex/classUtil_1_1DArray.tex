class Util\+::\+D\+Array$<$ Data $>$}

Dynamically allocatable contiguous array template. 

A \mbox{\hyperlink{classUtil_1_1DArray}{D\+Array}} wraps a dynamically allocated C \mbox{\hyperlink{classUtil_1_1Array}{Array}}, and stores the size of the array. A \mbox{\hyperlink{classUtil_1_1DArray}{D\+Array}} can be allocated, deallocated or reallocated (i.\+e., resized and moved) by member functions.\+j

The Array$<$\+Data$>$ base class provides bounds checking when compiled in debug mode.
