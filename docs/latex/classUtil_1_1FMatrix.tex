\subsubsection*{template$<$typename Data, int M, int N$>$\newline
class Util\+::\+F\+Matrix$<$ Data, M, N $>$}

Fixed Size \mbox{\hyperlink{classUtil_1_1Matrix}{Matrix}}. 

The \mbox{\hyperlink{classUtil_1_1FMatrix}{F\+Matrix}} class wraps a statically allocated 1D C array, but provides access to its elements via the A(i,j) \mbox{\hyperlink{classUtil_1_1Matrix}{Matrix}} syntax.

Template parameters M and N are the number of rows and columns respectively, so that capacity1 = M and capacity2 = N.
