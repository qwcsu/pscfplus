\doxysubsubsection{Detailed Description}
A set of boolean variables represented by characters. 

A \mbox{\hyperlink{classUtil_1_1FlagSet}{Flag\+Set}} has a string of allowed characters, each of which which represents a boolean variable (i.\+e., a flag), and a string of actual characters containing the subset of the allowed characters that should be set on (i.\+e., true).
