\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+G\+Stack$<$ Data $>$}

An automatically growable Stack. 

A \mbox{\hyperlink{classUtil_1_1GStack}{G\+Stack}} is stack that is implemented as a growable pointer array. Like any pointer array it holds pointers to objects, rather than objects, and associated objects are not destroyed when the container is deallocated or destroyed.
