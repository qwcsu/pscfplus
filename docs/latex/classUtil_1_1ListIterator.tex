\subsubsection*{template$<$typename Data$>$\newline
class Util\+::\+List\+Iterator$<$ Data $>$}

Bidirectional iterator for a \mbox{\hyperlink{classUtil_1_1List}{List}}. 

A \mbox{\hyperlink{classUtil_1_1ListIterator}{List\+Iterator}} provides bidirectional input/output access to a linked list, similar to an S\+TL bidirectional iterator. An $\ast$ operator returns a reference to an associated Data object. The ++ and -- operators change the current pointer to the next or prev element in a list.

The \mbox{\hyperlink{classUtil_1_1ListIterator_ae34360ffdd3a750ac3eb735dc5eeeeec}{is\+End()}} method returns true if either end of the list has already been passed by a previous ++ or -- operation. When \mbox{\hyperlink{classUtil_1_1ListIterator_ae34360ffdd3a750ac3eb735dc5eeeeec}{is\+End()}} is true, the iterator is no longer usable, since it no longer points to a \mbox{\hyperlink{classUtil_1_1Node}{Node}} and cannot be incremented or decremented.
