\subsubsection*{template$<$typename Event$>$\newline
class Util\+::\+Notifier$<$ Event $>$}

Abstract template for a notifier (or subject) in the \mbox{\hyperlink{classUtil_1_1Observer}{Observer}} design pattern. 

In the observer design pattern, a \mbox{\hyperlink{classUtil_1_1Notifier}{Notifier}} manages a list of registered \mbox{\hyperlink{classUtil_1_1Observer}{Observer}} objects, and provides a method to notify all observers when some event occurs. A list of observer objects is maintained as a list of \mbox{\hyperlink{classUtil_1_1Observer}{Observer}} pointers. The method Notifier\+::notify\+Observers(\+Event\&) method calls the the update(\+Event\&) method of every \mbox{\hyperlink{classUtil_1_1Observer}{Observer}} in the list.

The typename parameter Event is the type of the object that must be passed to the update() method of each observer. This type can name either a primitive C data type or a class, but must encode whatever information is required for any \mbox{\hyperlink{classUtil_1_1Observer}{Observer}} to respond appropriately when notified.
