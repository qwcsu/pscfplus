\subsubsection*{template$<$typename Event$>$\newline
class Util\+::\+Observer$<$ Event $>$}

Abstract class template for observer in the observer design pattern. 

An \mbox{\hyperlink{classUtil_1_1Observer}{Observer}} is notified of an event by calling its update method. The template class parameter Event is the type of object that is passed to the \mbox{\hyperlink{classUtil_1_1Observer_a518cb0d0b98052c436d4a52c61edc80b}{update()}} method as a message about an event.
