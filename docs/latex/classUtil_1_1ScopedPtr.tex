\doxysubsubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Util\+::\+Scoped\+Ptr$<$ T $>$}

A very simple R\+A\+II pointer. 

A \mbox{\hyperlink{classUtil_1_1ScopedPtr}{Scoped\+Ptr}} mimics a built-\/in pointer, except that guarantees destruction of the object to which it points when the \mbox{\hyperlink{classUtil_1_1ScopedPtr}{Scoped\+Ptr}} goes out of scope. It accepts ownership of a built-\/in pointer either upon construction or by the \mbox{\hyperlink{classUtil_1_1ScopedPtr_acb03afa4ff6d00f458453ea7a5ddade1}{reset()}} method, and deletes the associated object in its destructor. A \mbox{\hyperlink{classUtil_1_1ScopedPtr}{Scoped\+Ptr}} cannot be copy constructed or assigned.

Similar to boost\+::scoped\+\_\+ptr, with minor differences. It takes the same amount of memory as a built-\/in pointer, and should be equally fast.
