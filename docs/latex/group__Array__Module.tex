\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{classUtil_1_1Array}{Array}} containers that store objects by value, and related iterators. 

The \mbox{\hyperlink{classUtil_1_1Array}{Array}} containers that do not have a P prefix are one-\/dimensional array containers that store objects by value. These all overload the subscript \mbox{[}\mbox{]} operator to provide access to elements as references.

The \mbox{\hyperlink{classUtil_1_1DArray}{D\+Array}} and \mbox{\hyperlink{classUtil_1_1FArray}{F\+Array}} containers are simple wrappers for dynamically allocated and fixed-\/size C arrays, respectively. The \mbox{\hyperlink{classUtil_1_1DSArray}{D\+S\+Array}} and \mbox{\hyperlink{classUtil_1_1FSArray}{F\+S\+Array}} containers are dynamically and statically allocated arrays, respectively, that have both a fixed capacity but a variable logical size, with contiguous elements. The \mbox{\hyperlink{classUtil_1_1GArray}{G\+Array}} container is a growable sized array, similar to a std\+::vector. Destructors for these arrays all delete the associated C array of objects.

An \mbox{\hyperlink{classUtil_1_1RArray}{R\+Array}} $<$ T $>$ is an \mbox{\hyperlink{classUtil_1_1Array}{Array}} that is intended to be used as an alias for, or a shallow copy of, a target \mbox{\hyperlink{classUtil_1_1DArray}{D\+Array}}, \mbox{\hyperlink{classUtil_1_1FArray}{F\+Array}} or C array. An \mbox{\hyperlink{classUtil_1_1RArray}{R\+Array}} contains a copy of the array address and capacity of the target array, where are copied by the \mbox{\hyperlink{classUtil_1_1RArray_a5ce7d0c2e141487a265591cc8e6d00dc}{R\+Array\+::associate()}} method. Like other array containers, an \mbox{\hyperlink{classUtil_1_1RArray}{R\+Array}} overloads the \mbox{[}\mbox{]} operator to provide access to elements as references. The destructor of an \mbox{\hyperlink{classUtil_1_1RArray}{R\+Array}} does not delete the associated C array.

A \mbox{\hyperlink{classUtil_1_1RingBuffer}{Ring\+Buffer}} is a cylic buffer array for which the append() method adds elements to the end of a sequence if the buffer is not full, and overwrites the oldest element if it is.
